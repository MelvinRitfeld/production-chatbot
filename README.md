# Production Chatbot

## Bescrijving
Dit project is een productieklare AI-chatbot applicatie gebouwd met een moderne full-stack architectuur.  
Het doel van dit project is om een schaalbare en onderhoudbare chatbot te ontwikkelen met behulp van Docker, een backend API en een database.

## Tech stack
- **Frontend:** Next.js
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL
- **Containerisatie:** Docker & Docker Compose
- **AI Integratie:** OpenAI API (LLM)
